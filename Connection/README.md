**BigQuery–Colab Workflow**

This folder demonstrates how to connect Google BigQuery with Google Colab to execute SQL queries on cloud-hosted datasets and load the results into Python for analysis.

**What this notebook demonstrates**
This notebook shows how to:
    - authenticate access between Colab and BigQuery
    - connect to a specific cloud project
    - execute SQL queries on a dataset
    - convert query results into a Pandas DataFrame
    - preview dataset output inside Colab

    
**Purpose of this workflow**
This workflow helps build practical experience with:
    - cloud-based SQL querying
    - Python notebook environments
    - structured dataset access
    - reproducible analytics workflows
It forms part of my learning path toward data analytics and machine learning pipeline development.
